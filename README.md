# ក្រុមសន្សំប្រាក់អនាគតយើង - Saving For Our Future (SOF)

## មុខងារគន្លឹះរបស់ប្រព័ន្ធ (Key Features)

1. **📱 អេក្រង់ទូរស័ព្ទ (Fully Mobile Match):** រាល់មុខងារ ប៊ូតុង តារាង (Tables with horizontal scroll) និងទម្រង់បញ្ចូលទិន្នន័យទាំងអស់ត្រូវបានកែសម្រួលដិតដល់ឱ្យសមស្របទៅនឹងអេក្រង់ទូរស័ព្ទដៃ ដោយលុបបំបាត់បញ្ហាលេច និងធ្លាយទិន្នន័យ។
2. **📊 ផ្ទាំងគ្រប់គ្រងទូទៅ (Dashboard):** បង្ហាញសាច់ប្រាក់សរុប សមាជិកសរុប ប្រាក់សន្សំសរុប តុល្យភាពកម្ចី និងប្រាក់ចំណេញពីការប្រាក់ ព្រមទាំងតារាង Recharts វិភាគចរន្តសាច់ប្រាក់។
3. **👥 គ្រប់គ្រងសមាជិក (Member Management):** ចុះឈ្មោះ និងកែសម្រួលប្រវត្តិរូបសមាជិក បង្កើត ID សមាជិកដោយស្វ័យប្រវត្តិ។
4. **💰 ការសន្សំប្រាក់ (Savings Module):** កត់ត្រាការដាក់ប្រាក់សន្សំ ដកប្រាក់សន្សំ និងគណនាការប្រាក់សន្និរំលាយប្រចាំខែ។
5. **🏦 គណនីកម្ចីប្រាក់ (Loans Module):** បង្កើតគម្រោងឥណទានថ្មី បញ្ចូលអត្រាការប្រាក់ កត់ត្រាការបង់ប្រាក់សងទាំងការ និងការប្រាក់។
6. **📝 របាយការណ៍សង្ខេប (Reports & History):** រៀបចំរបាយការណ៍ប្រចាំខែ នាំចេញជាទម្រង់ Excel/CSV និងជួយសម្រួលក្នុងកម្មវិធីប្រជុំ។

## របៀបដំឡើង និងដំណើរការគម្រោងឡើងវិញ (Installation Guidance)

គម្រោងនេះដំណើរការលើ Node.js បរិស្ថាន React និង Vite យ៉ាងរហ័ស៖

```bash
# ១. ទាញយកគម្រោងពី GitHub
git clone https://github.com/sophakcamkids-gif/sofmanagementapp.git
cd sofmanagementapp

# ២. ដំឡើងកញ្ចប់មេ (Packages Installation)
npm install

# ៣. ដំណើរការនៅក្នុងម៉ាស៊ីន (Local Dev Mode)
npm run dev
```
